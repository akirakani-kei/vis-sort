## Installation (UNIX)

Precompiled:
```
git clone https://akirakani-kei/vis-sort \
&& cd vis-sort \
&& chmod +x vis-sort.out
```
*Run via:*
```
./vis-sort.out
```

***OR***

Self compile:
```
git clone https://akirakani-kei/vis-sort \
&& cd vis-sort \
&& g++ vis-sort.cpp
```
*Run via:*
```
./a.out
```
